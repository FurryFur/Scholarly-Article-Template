\subsection{Objective Function}
We use WGAN-GP as our objective objective function [reference]. The original WGAN-GP formula is defined as:
\begin{equation}
\underset{\tilde{x} \sim P_g}{\mathbb{E}}[D(\tilde{x})] - \underset{x \sim P_r}{\mathbb{E}}[D(x)] + \lambda \underset{\hat{x} \sim P_g}{\mathbb{E}}[D(\tilde{x})] - \underset{x \sim P_r}{\mathbb{E}}[D(x)]
\end{equation}
